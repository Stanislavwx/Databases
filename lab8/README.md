# Clients --- Python Client for PostgreSQL Database

## Опис

Це консольний клієнт для взаємодії з базою даних PostgreSQL, розроблений
у рамках лабораторної роботи №8.\
Додаток працює через прямі SQL-запити та реалізує основні CRUD-операції
над таблицею **clients**.

## Функціональність

Клієнт дозволяє:

-   переглядати список таблиць у схемі *public*;
-   переглядати структуру конкретної таблиці;
-   виконувати операції:
    -   **SELECT** (виведення всіх клієнтів),
    -   **INSERT** (додавання нового клієнта),
    -   **UPDATE** (оновлення клієнта),
    -   **DELETE** (видалення клієнта);
-   виконувати довільні SQL-запити типів SELECT/INSERT/UPDATE/DELETE;
-   обробляти помилки СУБД та виводити діагностику.

## Вимоги

-   Python 3.10+

-   PostgreSQL 15+

-   Встановлений Python-пакет:

        pip install psycopg2-binary

## Налаштування підключення

У файлі `client.py` змінити параметри підключення за потреби:

``` python
DB_CONFIG = {
    "host": "127.0.0.1",
    "port": 5432,
    "dbname": "lab8db",
    "user": "labuser",
    "password": "labpass",
}
```

## Запуск

1.  Активувати віртуальне середовище (якщо використовується):

        source venv/bin/activate

2.  Запустити клієнт:

        python client.py

## Структура таблиці `clients`

  Поле         Тип            Опис
  ------------ -------------- ----------------------------
  id           SERIAL         Первинний ключ
  name         VARCHAR(100)   Ім'я клієнта
  email        VARCHAR(100)   Унікальна електронна пошта
  age          INTEGER        Вік клієнта
  created_at   TIMESTAMP      Дата створення запису
